V2.1已经发布

发表于 2017-05-17   |   分类于 版本发布

本次主要更新

时隔这么久了，终于发布了一个小版本。本次更新主要包含以下几点：

  • 导航栏去掉『排行榜』菜单入口，添加『此刻』菜单
  • 添加评论功能
  • 完善『我的订阅』页面
  • 修复一些 BUG

关于『此刻』页面说明

『此刻』页面显示注册用户订阅动态，按最新时间排序。可以方便大家获取其他小伙伴在关注哪些应用和电子书。

关于『评论』功能说明

评论目前只是一个尝试功能，目前只能评论，不能回复他人评论。欢迎大家多使用和反馈。
