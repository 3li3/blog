V3.0 已经发布

发表于 2018-09-02   |   分类于 版本发布

最近半个月，三立三主要做了以下更新：

  • 优化了网站顶部导航菜单样式，让菜单在手机上使用更方便。
  • App 和 电子书导航默认为『今天最值』，显示内容为经过一个简单算法的当天降价消息（算是重大更新了）。
  • 推出了会员计划，简单来说就是普通会员只能免费订阅10个『商品』，详情参考三立三会员页面。
  • 优化了电子书列表的样式，并且添加了一些筛选条件，让用户更方便的找到自己想要的电子书。

最后，感谢您的关注和支持。
